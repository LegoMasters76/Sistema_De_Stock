#MiprimerpaginaMandarini


Sistema de Stock desarrollado con Django

#Funiconalidades

1-Pagina de inicio
2-Ver Stock
	Busqueda de productos
	Editar productos
	Eliminar productos
	Agregar productos

3-Agregar productos
	Nombre
	Descripcion
	Precio
	Stock
	Categoria
	Provedor

4- Nueva categoria
	Nombre
	Descripcion

5-Nuevo Provedor
	Nombre
	Telefono
	Email

6-Buscador de Productos
	Sistema de filtro por nombre del producto



# Modelos

1- Producto
2- Categoria
3- Proveedor

# Como probar

1- Ejecutar servidor

python manage.py runserver

2- Entrar al siguiente enlace

http://127.0.0.1:8000
